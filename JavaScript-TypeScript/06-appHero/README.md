# APP HEROES

# levantar desarrollo

nota: asegurate de que el backend este levantado es el proyecto: 06-nest-heroes-backend

1.- clonar el repositorio
2.- colocar las variables de entorno de acuerdo al archivo .env_templates
3.- correr el siguiente comando en la terminal `npm install`
4.- correr la aplicacion con el siguiente comando `npm run dev`
